# SH Budget App - Frontend

공유 가계부 서비스의 모바일/웹 클라이언트 앱입니다.

## 기술 스택

| 구분 | 기술 |
|------|------|
| **Framework** | React Native + Expo SDK 54 |
| **Language** | TypeScript |
| **라우팅** | Expo Router v6 (파일 기반) |
| **스타일링** | NativeWind v4 (Tailwind CSS) |
| **상태 관리** | Zustand v5 |
| **데이터 페칭** | TanStack React Query v5 |
| **HTTP 클라이언트** | Axios |
| **폼/검증** | React Hook Form + Zod |
| **보안 저장소** | expo-secure-store |

## 주요 화면

### 1. 홈 - 거래내역
- **캘린더 뷰**: 월별 캘린더에 일별 수입/지출/이체 표시, 한국 공휴일 지원
- **리스트 뷰**: 날짜별 거래 내역 그룹핑
- **월별 요약**: 총 수입, 총 지출, 순수입 카드
- 캘린더/리스트 토글, 당겨서 새로고침

### 2. 자산 관리
- 총 자산 카드 (자산/부채 분리)
- 자산별 월별 거래 내역 조회
- 자산 추가/수정/삭제

### 3. 통계
- 월별 수입/지출 요약 카드
- 수입 vs 지출 막대 차트
- 카테고리별 지출 비율 (프로그레스 바)
- 멤버별 기여도 (2인 이상일 때)

### 4. 설정
- 가계부 선택 및 이름 수정
- 초대 코드 복사 (가계부 공유)
- 멤버 목록 (OWNER/MEMBER)
- 카테고리 관리 (추가/수정/삭제)
- 반복 거래 관리
- 푸시 알림 설정 (웹 전용)
- 다른 가계부 참여 (초대 코드 입력)

### 5. 거래 추가
- 지출/수입/이체 타입 선택
- 커스텀 숫자 키패드
- 날짜 선택 (캘린더 모달)
- 자산/카테고리 선택
- 반복 거래 설정 (매월 N일)

## 로컬 개발 환경 설정

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start

# 플랫폼별 실행
npm run ios       # iOS 시뮬레이터
npm run android   # Android 에뮬레이터
npm run web       # 웹 브라우저
```

### 웹 빌드 (PWA)

```bash
npm run build:web
```

### 백엔드 연결

- **웹 개발**: `http://localhost:8080`
- **모바일 개발**: `http://<로컬IP>:8080`
- **프로덕션**: `https://shbudgetbe-production.up.railway.app`

API 클라이언트에서 환경에 따라 자동으로 URL이 결정됩니다 (`src/api/client.ts`).
